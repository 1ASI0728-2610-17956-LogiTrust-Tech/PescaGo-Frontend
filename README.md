# PescaGo Frontend

Interfaz desarrollada con Angular 19.

## 1. Instalación de dependencias

1. **Node.js**:
   - Descarga e instala [Node.js](https://nodejs.org/) (versión 18 o superior).
   - Verifica la instalación con: `node -v`.

2. **Instalar paquetes**:
   - Navega a la carpeta del Frontend, abre una terminal y ejecuta:
     `npm install`

## 2. Ejecución

Una vez instaladas las dependencias, inicia el servidor de desarrollo:
`npm start`

La aplicación estará disponible en http://localhost:4200.

## 3. Solución de problemas comunes

- **Error de conexión con el backend**: Asegúrate de que el backend esté corriendo en el puerto 8080 antes de levantar el frontend.
- **Error de permisos en npm**: Si recibes errores de acceso, intenta abrir la terminal como administrador o ejecuta `npm cache clean --force`.
